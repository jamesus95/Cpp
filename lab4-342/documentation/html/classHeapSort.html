Helper subroutine to help re-structure the heap after max value is extracted. :

  1. Determine whether to swap down the left or right
  2. Recursively swap down accordingly
    Condition for swapping left/right:
  3. left/right child must be greater than parent
  4. swap left if left child >= right child
  5. swap right if left child < right child

indexindex of parent value in the array
sizesize of the array, right child cannot exceed

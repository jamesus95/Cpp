Performs sorted-merge on two linked lists via manipulating pointers. Creates only O(1) extra storage Nodes.

Operates on the same logic as merge sort: add the smaller of the two current values to the merger, then advance and repeat. Once either source is exhausted, append the rest of the other to merger.

)
private

Recursive routine to perform mergesort on a Linked List starting at subRoot and spanning up to size == size.

This routine manipulates pointers for sorting. It creates O(1) extra storage space for each split & merge.
Therefore the total extra space created is O(logn).
Otherwise, it operates on the same logic as recursive merge sort on arrays.

Parameters
subRootpointer to the front of the linked list
sizesize of the linked list

pointer to the front of the sorted linked list

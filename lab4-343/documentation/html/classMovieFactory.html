Detailed Description

Purpose: Act as a centralized generator for various Movie subtypes (Classic, Comedy, Drama).

Given a code, the MovieFactory will return an empty instance of the Movie subtype indicated by that code (NULL if the subtype is not supported).

Inherit from: ItemFactory (even though it doesn't really need to) Inherited by: <none>

Detailed Description

Purpose: Act as a centralized generator for various Transaction subtypes. E.g. Borrow, Return, Display User, Display Inventory

Given a code, the TransactionFactory will return an empty instance of the Transaction subtype indicated by that code (NULL if the subtype is not supported).

Inherit from: ItemFactory (not really necessary) Inherited by: <none>

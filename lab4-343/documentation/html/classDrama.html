void Drama::parseTransactionData ( istream &  in)
virtual

Extract info from the input stream and parses it into the private fields.

Uses the getter subroutines to parse the data one field at a time The subroutines must be used in a correct order for a particular data format. This routine is called when initializing the transactions. The presumed format for input is: director, movie title (both of which are comma-separated)

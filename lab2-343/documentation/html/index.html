Binary Search Tree

Introduction

A dynamic, list-based implementation of the Binary Search Tree data structure. The tree stores and manages the data object from the client via pointers.

Assumptions

Data class must have operator==, operator>, operator!=, operator<<

Notes

Warning
The tree class does not support remove() yet. One workaround is to move all the data to an array, remove / overwrite the desired target, then rebuild the tree. This won't preserve the tree's structure, as the new tree will be balanced.
The tree <-> array transfer routines do not support arrays holding more than 100 items.

bool TBST::removeRoot ( )
private

helper method for remove(). Focuses on removing the root Node in various circumstances.

There are 4 situations in which a root Node is to be removed:

  1. Root is the only node in the tree
  2. Root has a left branch
  3. Root has a right branch
  4. Root has both children branches

int  order

functional programming implementation of BST recursive traversal

This method traverses the tree starting at the Node specified, applying the function func specified, in the order specified.

Precondition
If no order is specified or if specified order is invalid, traverses in-order.
Parameters
funcfunction to apply to each Node pointer along the traversal path not check input. YOLO. Design this function with only a single Node pointer in mind.
outputostream object to append the output content to
orderorder of recursive traversal (in-order = 0, pre-order = 1, post-order = -1)
